Attain Alarm Management

Attain includes full alarm event management. Alarms include any fault detection & diagnostics event or critical problem. Fault detection and diagnostics logic is applied to each device automatically with no end user configuration required. Any device can be configured for critical alert notification

Alarm Interfaces

    Alarm page contains :

  • Tabulated list of fault detection & diagnostics events
  • Event information (timestamp, description, severity)
  • More event detail (3 dots - details column)
  • Additional event handling logic can assign events to platform users
  • Alarm severity can be modified to suit customer requirements
