Welcome to the Attain Web Interface (AWI), an easy to use REST API enabling seamless integration with your Attain platform

Note : For white label customers / tenant administrators only. Replace <...> with your project specfic information

    Notes

  • Project entities are split into two core categories: 1.Devices and 2.Assets. Devices include all MEP equipment. Assets include buildings and each floor or main demise within each building
  • Each project entity has two data types: 1.Static/semi-static and 2.Timeseries. Follow guidance below, and accompanying pages to access entities data
  • Static/semi-static data are data points that dont change or change infrequently. This could be a design parameter or a parameter that gets logged once per month
  • Timeseries data are data points that change frequently and are recorded at fixed time intervals, typically every 15 mintes. An example would be electrical power from a power meter

    • Other Notes

  • When aggregating data from multiple buildings to a specific entity, use the site name, format (see below) : CUSTOMER NAME-SITE NAME

  • Data Access Guide

  • 1. Authenticate your user and obtain access token here : https://<host>/api/auth/login , and periodically refresh the token here : https://<host>/api/auth/token , All subsequent requests will require Bearer token in request header
  • 2. Fetch site details. Each site is represented as PROJECT_ABCD, where ABCD is a unique project reference : https://<host>/api/customers?pageSize=100&page=0 , Site name can be found under : payload.data[index].additionalInfo.description, Save the project ID for the project you wish to access
  • 3. Fetch project devices (MEP devices) detail : https://<host>/api/customer/<project ID>/devices?pageSize=500&page=0 , Save the device ID for the device(s) you wish to access
  • 4. Fetch project assets (building/floors) detail : https://<host>/api/customer/<project ID>/assets?pageSize=50&page=0 , Save the asset ID for the asset(s) you wish to access
  • 5. Use 'Devices Data' or 'Asset Data' guides to access device or asset data for the selected project
