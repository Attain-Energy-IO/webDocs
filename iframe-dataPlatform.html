    API Access

  1. Project entities are split into two core categories: 1.Devices and 2.Assets. Devices include all MEP equipment. Assets include buildings and each floor or main demise within each building
  2. Each project entity has two data types: 1.Static/semi-static ATTRIBUTES and 2.Timeseries TELEMETRY. Entities are mapped to Entities in the navigation menu
  3. Static/semi-static data are data points that dont change or change infrequently. This could be a design parameter or a parameter that gets logged once per month
  4. Timeseries data are data points that change frequently and are recorded at fixed time intervals, typically every 15 mintes. An example would be electrical power from a power meter
