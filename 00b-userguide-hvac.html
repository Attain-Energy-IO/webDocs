Dashboard Group 3 – HVAC System

The HVAC System dashboard group presents information on building systems including - heating services, cooling services, packaged AC, critical cooling, ventilation, domestic hot/cold water

3.0 - Overview

    Dashboard contains :

  • Top row cards with main HVAC equipment status counts
  • 30 day chart of building thermal performance

    Target characteristics :

  • Minimal concurrent blue and red portions. Red (heat) portions in winter, Blue (cool) portions in summer
  • Daily thermal output moderation (days with less red and blue)
  • Heat (red) or Cool (blue) maxima falls within green demand area

3.1 - Floors / Tenants drilldown

Clicking on Tenants button at top of overview page navigates to a floors / tenants listing table :

    Dashboard contains :

  • Tenant / Floor list with count of HVAC zones in heating (red) or cooling (blue)
  • Average air quality and,or temperature per tenant / floor

    Target characteristics :

  • Higher red (heat) count in winter and blue (cool) count in summer
  • Average CO2 level less than 800PPM
  • Average space temperature between 20 and 24 Deg C

3.1.1 - Floor / Tenant equipment listing

Clicking on a tenant row on above table navigates to a tenant HVAC equipment list :

Dashboard contains:

  • HVAC equipment operating status - Key realtime information per unit

Target characteristics:

  • Normal expected point values
  • Normal runtime cycles (hours since ON not excessive)
  • Equipment healthy (no critical fault indication)
