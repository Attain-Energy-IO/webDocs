Attain VBM - Workspace Management Web App

VBM provides building tenants with enriched workspace information, slick management functions and full workspace control

The web-app uniquely integrates with Attain's Active-Trim system enabling better workspace control and improved building energy performance

App Features Include:

  1. Temperature view / adjustment
  2. Ventilation view / adjustment
  3. Lighting view / adjustment
  4. Workspace time scheduling
  5. FM ticketing system

Comfort Indicator

When browsing to a zone, a colour-coded comfort indicator is shown:

Comfort is calculated as the aggregate of all available zone data e.g. thermal, air quality, lighting, etc.

The number represents comfort from 10 (excellent comfort) to 0 (poor comfort)

A traffic-light like colour scheme represents area comfort
