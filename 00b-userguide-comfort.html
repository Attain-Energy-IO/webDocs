The Comfort dashboard group presents information on building comfort levels

4.0 - Overview

    Dashboard contains :

  • Top chart with 30 days overall building environmental comfort score
  • Bottom chart with 30 days log of workspace management app adjustment requests

    Target characteristics :

  • Continuous high comfort score (top chart)
  • Minimal workspace management adjustment requests

4.1 - Floors / Tenants environmental listing

Clicking on ANALYSIS button at top of Running Building Comfort Level chart navigates to a floors / tenants list :

    Dashboard contains :

  • Tenant / Floor list with average space temperature and average CO2 concentration
  • Traffic light scoring for thermal comfort and air quality levels

    Target characteristics :

  • Green indication for Thermal Comfort and AQ Comfort

4.1.1 - Floor / Tenant environmental report

Clicking on a tenant row on above table navigates to a tenant environmental report :

Dashboard contains:

  • Key environmental factors from available sensors
  • List of dedicated environmental sensor units installed in area (bottom card)

Target characteristics:

  • Point values within thresholds
  • Green indication for tabulated factors
