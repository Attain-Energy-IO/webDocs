Introduction

This document acts as a user guide for the Attain Energy platform user interface. Each connected building has a set of dashboard pages available once logged in.

Every effort is made to provide a harmonized and consistent view of buildings connected to Attain. Due to variability in configuration and data, some differences between dashboards may occur.

User Roles

Users are assigned access privileges upon account creation. They fall into one of the following categories:

  • Estate User — Access to many or all buildings in an estate
  • Building User — Access to a single building
  • Floors/Tenant User — Access to one or more floors in a building

Dashboard Groups
