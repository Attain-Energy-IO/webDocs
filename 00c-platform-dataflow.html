This guide specifies how data is transferred between sites and the Attain platform. The platform uses simplified data structures to enable efficient & extensible data transmission

  • Enablement notes for site systems configuration
  • Equipment with operatingMode telemetry, to be mapped to prescrobed numeric value for all possible equipment modes i.e. 0=OFF,1=AUTO,2=HEAT,3=COOL,4=FAN
  • Suitable equipment to have writable operatingMode. Local control logic will handle setting equipment to required mode
  • Equipment to have writable primary control setpoint. Setpoint types are temperature, flow, speed, capacity, etc
  • Equipment to have time scheduling capability
  • Floors or Tenants shall have writable floor settings

  • Site telemetry data object | SITE (gateway) PUSH > PLATFORM | 15 minute intervals

  • All device metadata and relations are stored as server side device attributes
