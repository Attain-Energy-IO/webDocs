Dashboard Group 10 – Reporting

Overview

User-selectable reports can be added to a customer instance on request. Reports can be accessed on-demand via the normal dashboard navigation menu, and can be configured for automatic transmission via email on any required schedule.

Optional Reports

Report Name Description
Runtime Report Report showing equipment scheduled operation
Critical Fault Report Report of critical faults
General Faults Report Report showing general faults
WELL Compliance Report showing information applicable to WELL standard

Device Reports

Reports can also include device-specific or system-specific outputs, such as:

  • DNCO
  • VRF
  • FCU
