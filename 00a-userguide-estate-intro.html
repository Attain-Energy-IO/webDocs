Introduction

This document acts as a user guide for the Attain Energy platform user interface. Each connected building has a set of dashboard pages available to users once logged in.

Every effort is made to provide a harmonized and consistent view of buildings connected to Attain. Due to high levels of variability in building configuration and data availability, some differences building-to-building should be expected

User Roles

Users are assigned access privileges upon account creation. They fall into one of the following categories:

  • Estate User — Access to many or all buildings in an estate
  • Building User — Access to a single building
  • Floors/Tenant User — Access to one or more floors in a building

Dashboard Groups

Dashboard groups categorize data into logical themes (Performance, Services, HVAC, etc.), each containing multiple dashboard pages for deeper drilldowns.

Group NameDescription
PerformanceOverall building performance
ServicesPrimary services & utilities
HVACHeating ventilation & air conditioning services
