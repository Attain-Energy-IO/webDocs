Attain Edge Connectivity

The Attain edge stack can be cloud or edge device hosted. We ship the Siemens Advanced IOT2050 or Dell Gateway 3200, or can load the software onto existing hardware such as a BMS head-end

The edge stack is responsible for providing data access between sites and cloud services. The gateway application handles all data sorting i.e. device/point naming

Network interfaces can be configured to handle multiple protocols. All configuration is via simple web UI

Security & Connectivity Features

  • Internet dataflow is northbound only HTTPS

  • Optional serial bridge configuration (1GW internet access/1GW BMS LAN access/linked via serial)
  • Optional dedicated 4G connectivity
