Dashboard Group 2 – Services

The services dashboard group presents utility usage (electricity,water,gas) for the building

Overview

The services overview page includes recent consumption data for each primary service:

Services Overview

This group presents primary utility services (electricity, gas, water) usage for the building. The overview page contains 1 month of building consumption for each service:

[Image: Services Overview Chart]

Aggregations shown include minimum, maximum, and average consumption for the selected interval.

Utility Analysis

Clicking a specific utility drills down to consumption analysis for the selected service:

[Image: Utility Consumption Analysis]
  • Bar chart: Previous months vs same month last year
  • Bar chart: Current month vs same month last year with month target
  • Line chart: Yesterday’s consumption with day target
  • Line chart: Last 7 days (Mon-Sun) with week target

Tenants & Landlord Analysis

From the utility analysis page, clicking 'Tenants' or 'Landlord' shows tabulated lists of utility supplies:

[Image: Tenant Utility Supply Table]
[Image: Landlord Utility Supply Table]

Clicking a row navigates to a page for the selected supply:

[Image: Tenant/Landlord Supply Details]
  • Bar chart: Up to 3 years previous monthly consumption
  • Line chart: Last 30 days consumption with min, max, average aggregation
