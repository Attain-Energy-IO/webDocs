Management API

    Quick API Access - Assets & Devices

  1. Authenticate your user and obtain access token here : https://<host>/api/auth/login , and periodically refresh the token here : https://<host>/api/auth/token , All subsequent requests will require Bearer token in request header
  2. Fetch project details. Each site is represented as SITE_NAME PROJECT_ABCD, where ABCD is a unique project reference : https://<host>/api/customers?pageSize=100&page=0
  3. Fetch project devices (MEP devices) detail : https://<host>/api/customer/<project ID>/devices?pageSize=500&page=0 , Save the device ID for the device(s) you wish to access
  4. Fetch project assets (building/floors) detail : https://<host>/api/customer/<project ID>/assets?pageSize=50&page=0 , Save the asset ID for the asset(s) you wish to access

  5. Device Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/timeseries
  6. Device Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/attributes
  7. Device Timeseries Point Data - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/values/timeseries?keys=<comma separated keys>&startTs=<range start UTC Timestamp milliseconds>&endTs=<range stop UTC Timestamp milliseconds> (Not including startTs and endTs results in last telemetry value being returned)
  8. Device Semi-Static Point Data - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/values/attributes?keys=<comma separated keys>

  9. Asset Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/timeseries
  10. Asset Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/attributes
  11. Asset Timeseries Point Data - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/values/timeseries?keys=<comma separated keys>&startTs=<range start UTC Timestamp milliseconds>&endTs=<range stop UTC Timestamp milliseconds> (Not including startTs and endTs results in last telemetry value being returned)
  12. Asset Semi-Static Point Data - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/values/attributes?keys=<comma separated keys>

Site data transports specify how data is transmitted between site and the Attain platform. The platform uses simplified data structures to enable easy transmision of telemetry and command/control data

The Attain data transport design ensures maximum data efficiency and enables other outcomes such as rapid site onboarding and fast add-on with other apps and services

  • Site telemetry data object | SITE -- PUSH --> PLATFORM | 15 minute intervals
