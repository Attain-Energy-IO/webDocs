Management API - Quick Access

Client Authentication

  1. Authenticate your user and obtain access token here : https://<host>/api/auth/login , and periodically refresh the token here : https://<host>/api/auth/token , All subsequent requests will require Bearer token in request header
  2. Fetch project details. Each site is represented as SITE_NAME PROJECT_ABCD, where ABCD is a unique project reference : https://<host>/api/customers?pageSize=100&page=0
  3. Fetch project devices (MEP devices) detail : https://<host>/api/customer/<project ID>/devices?pageSize=500&page=0 , Save the device ID for the device(s) you wish to access
  4. Fetch project assets (building/floors) detail : https://<host>/api/customer/<project ID>/assets?pageSize=50&page=0 , Save the asset ID for the asset(s) you wish to access

Device Quick Data Access

  • Device Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/timeseries
  • Device Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/attributes
  • Device Timeseries Point Data - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/values/timeseries?keys=<comma separated keys>&startTs=<range start UTC Timestamp milliseconds>&endTs=<range stop UTC Timestamp milliseconds> (Not including startTs and endTs results in last telemetry value being returned)
  • Device Semi-Static Point Data - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/values/attributes?keys=<comma separated keys>

Asset Quick Data Access

  • Asset Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/timeseries
  • Asset Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/attributes
  • Asset Timeseries Point Data - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/values/timeseries?keys=<comma separated keys>&startTs=<range start UTC Timestamp milliseconds>&endTs=<range stop UTC Timestamp milliseconds> (Not including startTs and endTs results in last telemetry value being returned)
  • Asset Semi-Static Point Data - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/values/attributes?keys=<comma separated keys>
