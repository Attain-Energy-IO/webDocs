Attain Project Onboarding

Step 1 - Data Pipeline Provisioning - (duration : 1 day)

  • For sites with a PC connected to the building management network, this machine can be used to host the Attain edge software
  • For sites with no pre-existing suitable hardware, an Attain edge gateway must be installed to connect the site
  • The machine hosting the Attain edge software must have an internet connection. This can be provided by onsite connectivity or via independent 4G connection
  • A suitable location should be identified by the on-site team, with building management network access and 1 x 230Vac socket outlet

Edge Software Configuration

  • Device Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/timeseries
  • Device Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/DEVICE/<device ID>/keys/attributes

Asset Quick Data Access

  • Asset Timeseries Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/timeseries
  • Asset Semi-Static Keys - Endpoint : https://<host>/api/plugins/telemetry/ASSET/<asset ID>/keys/attributes
